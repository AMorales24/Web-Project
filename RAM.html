RAM

RAM is a computer part that keeps data for a limited amount of time (Fisher, 2019) following when it is read (Parson). It works at quick speeds (Ducpham, 2018) and can also be used to view the data that is kept in it (Ducpham, 2018). If a computer has more RAM, it can deal with an increased amount of data (Fisher, 2019).
